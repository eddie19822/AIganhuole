Yi Jian (易剪) helps creators turn long narration into segments → stock footage → TTS → export, using DashScope for text, Pexels/Pixabay for stock, Doubao TTS for voice, plus optional Jimeng when Volcengine IAM is configured. Sign-in and usage days are enforced by your backend; API keys remain local. Built with Electron, React, TypeScript, and Vite.
