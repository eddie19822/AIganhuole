易剪面向解说、口播类创作者，将长篇文稿整理为「分段口播 + 匹配画面 + 配音 + 导出」工作流。应用可结合阿里云大模型进行文案分段与检索词生成，从 Pexels、Pixabay 等来源检索免版权视频并预览，使用火山豆包完成语音合成，经内置流程导出带配音的成片，并支持导出剪映工程。使用即梦等 AI 文生视频时，可在应用内单独配置火山访问密钥。使用需登录账户，可用天数与积分规则以服务端为准；各平台 API 密钥仅保存在本机。技术实现为基于 Electron 的桌面应用（React、TypeScript、Vite）。
